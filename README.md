# 📖 Project Description

The **Simple Calculator** is a browser-based application that allows users to perform basic mathematical calculations efficiently. The application is lightweight, responsive, and works without requiring any installation.

This project was created to strengthen front-end development skills and understand how JavaScript interacts with HTML elements to create dynamic web applications.

The calculator provides an intuitive interface where users can enter numbers, choose mathematical operators, and instantly receive accurate results.

# ⚙ Working Process

### Step 1

The user clicks any number button.

↓

### Step 2

JavaScript stores the selected value.

↓

### Step 3

The value appears on the calculator display.

↓

### Step 4

The user selects an arithmetic operator.

↓

### Step 5

The user enters another number.

↓

### Step 6

When "=" is pressed, JavaScript evaluates the expression.

↓

### Step 7

The calculated result is displayed instantly.

# 🎮 How to Use

1. Open the calculator.
2. Click the desired numbers.
3. Choose an arithmetic operator.
4. Enter the second number.
5. Click "=".
6. View the result.
7. Press **C** to clear everything.
8. Press **DEL** to delete the last entered digit.
